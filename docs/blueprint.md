# **App Name**: MediTrack

## Core Features:

- Appointment Scheduling: Allow users to schedule appointments with doctors and specialists.
- Medication Tracking: Enable users to track their medications, dosages, and schedules.
- Reminders and Notifications: Set up reminders for appointments and medication intake.
- Health Summary Generation: AI generates a summary of user's medication and appointments to serve as a tool to consult doctors

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) to create a sense of trust and reliability.
- Background color: Light gray (#F5F5F5) for a clean and modern look.
- Accent color: Teal (#008080) to highlight important actions and information.
- Body and headline font: 'PT Sans', a sans-serif that combines a modern look and a little warmth or personality, for clarity and readability.
- Use clear and recognizable icons to represent different functionalities and categories.
- Clean and intuitive layout with easy navigation for quick access to key features.
- Subtle animations and transitions to provide a smooth and engaging user experience.
